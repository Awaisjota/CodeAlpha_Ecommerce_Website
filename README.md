# 🛒 MERN E-Commerce Web App

A full-stack e-commerce web application built using **MERN Stack (MongoDB, Express, React, Node.js)** with Redux for state management.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login (JWT Auth)
- Browse Products
- Product Details Page
- Add to Cart / Remove from Cart
- Place Orders
- Responsive UI (Mobile + Desktop)

### 🛍️ Product Features
- Product Listing (Grid UI)
- Product Search & Filter
- Product Categories
- Stock Management
- Discount & Pricing System

### 🛠️ Admin Panel
- Admin Dashboard
- Manage Products (CRUD)
- Manage Orders
- Manage Users
- Product Form (Create / Edit)
- Order Status Management

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
